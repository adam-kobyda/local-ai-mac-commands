# local-ai-mac-commands
Speed up local AI workflow on Mac.

## Install

Clone the repo anywhere, then run:

```bash
bash install.sh
```

The script discovers every command in the repo and installs it to `~/.local/bin`. It will also offer to add `~/.local/bin` to your PATH if it isn't there already.

Re-run `install.sh` any time you pull new commands — it's safe to run multiple times.

## Commands

| Command | Description |
|---------|-------------|
| `last-downloaded` | Print the path of the most recently downloaded file in ~/Downloads |
